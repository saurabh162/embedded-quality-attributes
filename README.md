# Embedded Software Architecture by Quality Attributes

This repository accompanies my LinkedIn series on
Embedded Software Architecture.

The goal is to demonstrate practical architectural
techniques used to improve key quality attributes in
embedded systems through:

- Real embedded examples
- UML diagrams
- C++ implementations
- Unit tests
- Design trade-off discussions

---

## Quality Attributes

### ✅ Reliability
- HAL (Hardware Abstraction Layer)
- Retry Pattern
- Watchdog Pattern
- State Machines
- Heartbeat Monitoring

### ⬜ Maintainability
- SOLID Principles
- Layered Architecture
- Strategy Pattern
- Factory Pattern

### ⬜ Performance
- Memory Pools
- Static Allocation
- RTOS Design
- Event-Driven Architecture

### ⬜ Scalability
- Publish-Subscribe
- Plugin Architecture
- Feature Extension Techniques

### ⬜ Testability
- Dependency Injection
- Mocking
- Layered Testing
- Logging & Tracing
